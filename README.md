#Break Point is App written With FireBase Where User Can <br />
1 - Publish Posts in The Feed <br />
2 - Like Posts <br />
3 - Join Groups <br />
4 - create Groups And Add Members To Them <br />
5 - publish Posts In Group Seen Only By Members Of The Group <br />
